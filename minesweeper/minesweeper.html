Minesweeper

Instructions

Click a square to reveal if it contains a mine. When clicked, if a square does not contain a mine, the square will reveal the number of mines adjacent to the square. If the square contains no adjacent mines, all adjacent squares will be automatically clicked for you. Hold the shift key then click to flag a square you believe contains a mine. The game is won when all squares containing mines are marked with flags and every square not containing a mine is clicked. IIf you click a square containing a mine, you lose.
