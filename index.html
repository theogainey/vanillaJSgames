Theo's Vanilla JavaScript Games!

FreeCell

Played with a standard 52 card deck, this classic card game is won when the player successfully moves all cards to their respective foundation piles. Included in every release on Windows since 1995, what better game to build with vanilla JavaScript than FreeCell.

Right now, the game is playable but is missing a few key features. These missing features include but are not limited too a much-improved GUI.
